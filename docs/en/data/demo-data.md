# Demo Data Description

This project provides Demo data for teaching and toolchain validation. It is not a complete robotics dataset suitable for production training.

## Access

- Demo data site: [robodata.archebase.ai](https://robodata.archebase.ai)
- Synthetic examples in the repository: `docs/public/downloads/sample-episodes.jsonl`
- Data access request / collaboration form: [Submit an Application](https://archebase.feishu.cn/share/base/shrcnsMcwWICUVdNeutlRWhqyD5)

## What the GitHub Repository Contains

- [`episode.schema.json`](./schema/episode.schema.json): Episode data structure conventions;
- Synthetic JSONL examples containing no privacy-sensitive data from real robots;
- Data format validation scripts;
- Course documentation related to data collection, evaluation, and reproducibility.

The examples in the repository demonstrate field semantics, parsing workflows, and validation methods. Their scale, task coverage, and data distribution are not representative of ArcheBase's complete data assets.

## Usage Restrictions

1. Do not describe the Demo data as a complete or free dataset, or as a dataset that can be used directly for production training.
2. Do not infer the quantity, quality, or coverage of the actual data assets from the Demo data.
3. Licensing for the Demo data is managed separately from licensing for the course code.
4. For further data collaboration, contact ArcheBase through the application form.
